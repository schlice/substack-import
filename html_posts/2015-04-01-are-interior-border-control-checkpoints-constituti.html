Answer

{.rendered_qtext}
**Content:** [I\'m assuming you\'re talking about a national border and not a state border. State borders do not have border patrols.\ \ As long as the checkpoint is run by USCBP, they can place the actual checkpoint wherever they like. You\'d assume that they place it somewhere near the actual border, but it doesn\'t have to be directly at the border, it can be several miles in-country. They can have multiple checkpoints (at the border and further in-country), and all of this is Constitutional.]{.rendered_qtext}
