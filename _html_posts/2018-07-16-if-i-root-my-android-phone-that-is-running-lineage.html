**If I root my Android phone that is running LineageOS 15.1, is it
possible for a malicious app to gain root access without me being able
to deny it?**

The default root method is through ADB only, so apps running in the OS
can't gain direct access to root.

If you entered developer mode and applied root access to "Apps and ADB"
or "Apps" only, then apps can use root at will. Malicious apps can gain
root access without your knowledge.
