Answer









Content: [In logic, this is correct.\
\
A \--\> B\
B \--\> C\
\_\_\_\_\_\_\_\
therefore, A \--\> C\
\
This is valid logic. For example:\
\
Joe has gray hair.\
Gray hair looks old.\
Therefore, Joe looks old.\
\
The inverse is fallacious logic:\
\
A \--\> C\
B \--\> C\
\_\_\_\_\_\_\_\_\_\_\_\
therefore A \--\> B\
\
Example:\
\
Joe has gray hair.\
My dog has gray hair.\
Therefore, Joe is a dog.\
\
