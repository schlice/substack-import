--- title: '"How can I use game elements for commercial use?"' date: '2015-04-03' tags: - uncategorized layout: post --- Quora Answer 





Answer









Content: [In most cases, you would not be able to use characters
from copyrighted video games for commercial use.\
\
You would have to license the rights to use those characters from the
respective copyright holders.\
\
There are some \"fair uses\" that you can use without licensing, but
none of them are for-profit commercial uses.\
\
Contact the copyright holder to get permission to use the
character.]
